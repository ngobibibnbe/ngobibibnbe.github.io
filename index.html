PromptTrack

PromptTrack is a flexible and powerful library for object tracking using natural language prompts. It leverages advanced models such as MDETR and OWL-ViT V2 to perform robust visual grounding and identity-aware tracking, particularly in constrained or closed environments with fixed populations.

The project is open-source and actively maintained on GitHub. To make experimentation easier, we provide a ready-to-run Colab notebook and an intuitive Gradio interface on Hugging Face for visual testing — no setup required.

Identity aware tracking

We introduce an HMM-based framework for identity-aware long-term multi-object tracking, explicitly designed to handle sparse and uncertain re-identification cues in constrained animal monitoring settings. The implementation and evaluation code is available on GitHub.

PyTorch Hidden Markov Model Multi-object Tracking Animal Behavior Re-ID

Behavior analysis

We developed an unsupervised method to detect and localize abnormal behaviors in pig barns by leveraging spatio-temporal reconstruction errors from an autoencoder combined with tracking outputs. Anomalies are inferred when tracked pigs enter regions or exhibit patterns that the autoencoder fails to reconstruct reliably, allowing both spatial and temporal localization of novel activities.

PyTorch Autoencoder Anomaly Detection Spatio-Temporal Analysis Animal Behavior Tracking Unsupervised Learning

Spatio-Temporal action detection

I am building a spatio-temporal action detection pipeline by extending the InternVideo L-LVM architecture to generate action tubes and associate actions with individual entities across time. To enhance visual grounding and enable prompt-based localization, the system incorporates OWL-ViT for region-aware prompting and object-guided spatial proposals.
